# Signature — Приложение для захвата подписи

Маленькое Flutter-приложение для рисования цифровой подписи пальцем.

---

## Что делает

Пользователь рисует подпись на белом холсте, нажимает "Готово" —
подпись сохраняется как изображение PNG и отображается на главном экране.

---

## Структура

```
lib/
├── main.dart          # Точка входа
├── home_screen.dart   # Главный экран: показывает подпись или кнопку "Добавить"
└── signature_pad.dart # Холст для рисования подписи
```

## Поток навигации

```
main() → HomeScreen
           └── [кнопка "Добавить"] → SignatureScreen (рисование)
                 └── [кнопка "Готово"] → возврат на HomeScreen с PNG-байтами
```

---

## Известные проблемы

- Двойной MaterialApp в `main.dart` — нужно убрать вложенный MaterialApp из `_MyHomePageState`
- Подпись не сохраняется в файл — хранится только в памяти, пропадает при перезапуске
- Нет кнопки удалить/сбросить подпись на главном экране
- Release-сборка использует debug-ключ подписи
- Package name: `com.example.signature` — нужно менять перед публикацией
