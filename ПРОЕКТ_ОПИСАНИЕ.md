# Aura Scanner Project Overview

Updated: 2026-06-16

## What This Repository Contains

- `scanner_ap`: the main Flutter client for scanning, OCR, sync, and account flows
- `backend`: the Node.js and TypeScript API for auth and document storage
- `signature`: a smaller Flutter app dedicated to signature capture
- `scripts`: helper scripts for local verification

## Main Capabilities

- Email and social authentication
- JWT access and refresh token flows
- Document import and local storage
- Cloud document upload and synchronization
- OCR and translation helpers
- PDF utilities
- Signature capture
